Privacy Policy for InstinctShell

Last Updated: October 26, 2023

This Privacy Policy describes how InstinctShell ("the Application") handles information in connection with its use for managing VLESS/VMESS subscriptions and servers.

1. Data Collection and Storage

The Application does not collect, transmit, or store any personal data on external servers. All data remains locally on your device.

The following information is stored only on your device and is not accessible to the developer or any third parties:

  • VLESS/VMESS subscription links and configuration data
  • Server configurations and connection parameters
  • Application preferences and settings
  • Connection logs and usage statistics (if enabled in settings)

2. Data Processing

The Application processes subscription data and server configurations solely for the purpose of:

  • Establishing secure connections to your configured servers
  • Managing and updating subscription information
  • Providing connection statistics and performance metrics
  • Enhancing user experience within the application

3. Third-Party Services

InstinctShell interacts with the following third-party services:

  • Your Subscription Providers: The application fetches configuration data from the subscription links you provide. We are not responsible for the privacy practices of these external services.
  • Google Play Services: Used for application distribution and update mechanisms only.

4. Permissions

The Application requires the following permissions:

  • INTERNET: Required to connect to VPN servers and fetch subscription updates.
  • ACCESS_NETWORK_STATE: Used to check network availability and optimize connections.
  • VPN_SERVICE: Essential for establishing secure VPN connections.
  • FOREGROUND_SERVICE: Required to maintain active VPN connections in the background.
  • WAKE_LOCK: Prevents device from sleeping during active connections.

5. Data Security

All sensitive data (subscription links, server configurations) is stored locally using Android's secure storage mechanisms. The Application does not transmit this data to any external servers except to the destinations you explicitly configure.

6. User Rights

You have full control over your data:

  • You can add, modify, or remove subscription links at any time
  • All stored data can be completely erased by clearing the application data or uninstalling the application
  • You can export/import your configuration for backup purposes

7. Changes to This Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page and updating the "Last Updated" date.
